# scripts/experiments/ — exploratory analyses

One-off scripts used to investigate specific questions. They are not part of
the canonical flow and are kept for reference.

| Script | Question it explored |
|--------|----------------------|
| `eda_signal_fusion.py` | how axial and rotational channels relate |
| `experiment_baseline_scatter.py` | isolated baseline under LOEO, no tuning |
| `experiment_signal_branch.py` | FUSION vs axial-only vs rotational-only |

The conclusions from these now live in the main results (axial-only is the
strongest subset; the channels are largely redundant).
